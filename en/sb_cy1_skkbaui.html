What is サカキバラUI exactly?

A user interface powered by the Midori OS beta for the Nintendo 3DS/Nintendo Switch title "SmileBASIC" and "SmileBASIC 4" made by SmileBoom Co.Ltd. in 2015 and 2019 respectively.

This particular project of mine is different depending on which system you want to run it:

  • Nintendo 3DS version:
    The 3DS version is a mixture of a Windows-like desktop and the Nintendo DSi Menu for ease of use.
  • Nintendo Switch version (currently just a plan):
    The Switch version might be a window interface for the editor or direct mode which will be assisting you at the programming of your games and tools.
