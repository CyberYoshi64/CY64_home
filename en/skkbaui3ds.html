サカキバラUI for 3DS

サカキバラUI for 3DS is a mixture of a Windows desktop and the old Nintendo DSi menu.

4/27/2020: Notice: The 3DS version will be out of active development

On 20th July 2020, サカキバラUI for 3DS will be out of active development.
It will be out of active development because the development of the Nintendo Switch version will be started at that point.

After said date, you can still use the 3DS version but there may be a lack of updates for a very long period of time.

4/22/2020: Release of test build for old Nintendo 3DSs

サカキバラUI has been updated to run better on old members of the Nintendo 3DS family.

SmileBASIC 3 key for public test: QDQ4B3F3

3/27/2020: Release of version 0.3.6

サカキバラUI has been updated to add new functions to the desktop including new keyboard layouts & features

SmileBASIC 3 key: 51VVV3NJ

2/7/2020: Release of version 0.3.5r2

サカキバラUI has been updated to add the menu settings and the setup being made more like an update center.

1/26/2020: Release of version 0.3.5r1

サカキバラUI has been updated to fix crashges and bugs. Gotta clean some things...

SmileBASIC 3 key is no longer available.

2/7/2020: Release of version 0.3.5

サカキバラUI has been updated to add the サカキバラUI menu. It's pretty much the DSi Menu though.

Also, new keyboard layouts!
